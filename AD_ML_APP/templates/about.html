About This Project

The 🔬 Anaerobic Digestion ML App is an interactive machine learning platform designed to analyze, predict, and optimize biogas production parameters such as methane yield. This application integrates multiple machine learning models, feature engineering techniques, and optimization strategies into a single user-friendly dashboard.

It is specifically built to assist researchers, students, and engineers in understanding how different input parameters influence anaerobic digestion performance and to support data-driven decision making.

What This App Can Do

  • Upload and process custom CSV datasets
  • Train multiple ML models simultaneously
  • Compare models using R² Score and RMSE
  • Visualize performance using interactive graphs
  • Analyze feature importance for interpretability
  • Predict outputs based on user input
  • Download results and predictions as CSV files
  • Optimize input parameters for best output

Machine Learning Models Used

The app supports multiple regression algorithms:

  • Random Forest Regressor
  • Support Vector Regression (SVR)
  • Linear Regression
  • K-Nearest Neighbors (KNN)
  • XGBoost Regressor
  • Artificial Neural Network (ANN)

The best-performing model is automatically selected based on the highest R² score.

Model Evaluation Dashboard

The dashboard provides a complete comparison of model performance using:

  • R² Score comparison chart
  • RMSE comparison chart
  • Actual vs Predicted scatter plot

Interactive Plotly graphs allow zooming, hovering, and detailed inspection of results.

Feature Importance Analysis

The app identifies the most influential features affecting predictions using:

  • Tree-based feature importance (Random Forest, XGBoost)
  • Coefficient-based importance (Linear models)

This helps users understand which input variables have the greatest impact on the output.

Data Export Features

The application provides downloadable insights:

  • Full dataset with Actual vs Predicted values
  • Predictions with auto-filled missing inputs

This ensures transparency and allows further analysis outside the platform.

Optimization Module

The optimization feature determines the best input conditions to either:

  • Maximize output
  • Minimize output

It uses the trained best-performing model to explore input combinations and identify optimal operating conditions.

Technologies Used

  • Python (Flask Framework)
  • Scikit-learn & XGBoost
  • Pandas & NumPy
  • Matplotlib & Plotly
  • HTML, CSS (Jinja Templates)

Objective of the Project

The primary objective of this project is to bridge the gap between chemical engineering processes and modern machine learning techniques by creating an accessible, real-time analytical tool.

It demonstrates how ML can be applied to real-world engineering problems such as anaerobic digestion, process optimization, and performance prediction.

Final Note

This project showcases a complete machine learning pipeline — from data preprocessing and model training to evaluation, visualization, and optimization — all within an interactive web application.

It reflects the integration of engineering knowledge with modern AI tools to build practical, scalable solutions.
